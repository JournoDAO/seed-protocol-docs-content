# Seed

A **Seed** is the fundamental unit of data in Seed Protocol. It represents a single piece of content or data that can be published, versioned, and shared across apps and networks.

## Overview

Seeds are the building blocks of the protocol. Each Seed:

- Has a unique identifier (`seedUid`, `seedLocalId`)
- Belongs to a schema and model
- Can be versioned over time
- Is stored on permanent, decentralized infrastructure (Arweave) with attestations on Ethereum
