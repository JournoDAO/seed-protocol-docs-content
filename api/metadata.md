# Metadata

**Metadata** describes and enriches Seeds in Seed Protocol. It provides structured information about content—authorship, timestamps, schema references, and custom attributes—that enables discovery and interpretation.

## Overview

Metadata in Seed Protocol:

- **Schema-based** — Follows the model and property definitions of your schema
- **Machine-readable** — Enables semantic search and discovery
- **Flexible** — The querier determines meaning; metadata supports multiple interpretations

## Types of metadata

- **Provenance** — Who created it, when, and from what version
- **Schema** — Model name, property types, and structure
- **Attestations** — Ethereum-based attestations (e.g., via EAS) that persist alongside content
